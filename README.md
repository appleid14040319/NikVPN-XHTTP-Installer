# NikVPN XHTTP Installer

<div align="center">
  <img src="https://img.shields.io/badge/version-1.0.0-blue" alt="Version">

<br>

**یک اسکریپت تمام‌خودکار برای راه‌اندازی پروکسی VLESS+XHTTP+TLS با قابلیت مدیریت آسان از طریق پنل تحت وب.**

با این اسکریپت می‌توانید روی سرور اوبونتو یک پروکسی امن و پرسرعت راه‌اندازی کنید که آی‌پی اصلی شما را با استفاده از CDNهای رایگان (Vercel یا Netlify) مخفی می‌کند. همچنین می‌توانید چندین کانفیگ با UUIDهای مختلف و حجم مصرفی دلخواه بسازید و همه را از طریق یک پنل تحت وب مدیریت کنید.

---

## ✨ ویژگی‌های کلیدی

- **🚀 نصب کاملاً خودکار**: فقط دامنه را تنظیم کنید، بقیه کارها را اسکریپت انجام می‌دهد.
- **🔐 امنیت بالا**: ترافیک با TLS رمزگذاری می‌شود و آی‌پی سرور اصلی شما پشت CDN مخفی می‌ماند.
- **👥 چندین کانفیگ هم‌زمان**: در حین نصب، تعداد کانفیگ‌های مورد نیاز و حجم مصرف هر کدام را تعیین کنید.
- **🖥️ پنل تحت وب (3X-UI)**: مدیریت آسان کاربران، مشاهده مصرف، افزودن/حذف کانفیگ‌ها بدون نیاز به خط فرمان.
- **🌐 پشتیبانی از Vercel و Netlify**: انتخاب پلتفرم رله CDN به دلخواه شما.
- **🛠️ ابزار مدیریت خط‌فرمان (`nikvpn`)**: نمایش سریع کانفیگ‌ها، اطلاعات پنل، وضعیت سرویس و ...
- **🔄 تمدید خودکار SSL**: گواهی Let's Encrypt به‌طور خودکار تمدید می‌شود.
- **📊 تست سرتاسری**: پس از نصب، اتصال VLESS به‌طور کامل تست می‌شود و نتیجه نمایش داده می‌شود.

---

## 🏗️ معماری کلی

```
کلاینت (v2rayN/Xray) 
    ↕
Vercel/Netlify (Edge Function)
    ↕
سرور شما (Xray + 3X-UI Panel)
    ↕
اینترنت آزاد
```

- **کلاینت**: با لینک‌های تولید شده به رله CDN متصل می‌شود.
- **CDN (Vercel/Netlify)**: درخواست‌ها را به سرور اصلی شما ارسال می‌کند.
- **سرور اصلی**: Xray ترافیک را دریافت و پردازش می‌کند. پنل 3X-UI مدیریت کاربران را بر عهده دارد.

---

## 📋 پیش‌نیازها

قبل از شروع نصب، مطمئن شوید موارد زیر را در اختیار دارید:

| مورد | توضیح |
|------|-------|
| **سرور اوبونتو** | نسخه ۲۰.۰۴ یا بالاتر (توصیه: ۲۲.۰۴ یا ۲۴.۰۴) |
| **دامنه ثبت‌شده** | یک دامنه (یا ساب‌دامین) که رکورد A آن به آی‌پی سرور شما اشاره کند |
| **ایمیل معتبر** | برای دریافت گواهی SSL از Let's Encrypt |
| **توکن Vercel یا Netlify** | بسته به پلتفرمی که انتخاب می‌کنید |

### تهیه توکن‌ها:

- **Vercel**: به [تنظیمات توکن Vercel](https://vercel.com/account/tokens) بروید و یک توکن جدید بسازید.
- **Netlify**: به [تنظیمات توکن Netlify](https://app.netlify.com/user/applications#personal-access-tokens) بروید و یک Personal Access Token ایجاد کنید.

## 🚀 نصب سریع

برای نصب کافیست دستور زیر را در ترمینال سرور خود اجرا کنید:

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/nikvpn-iran/NikVPN-xhttp-installer/main/install.sh)
```

### مراحل نصب (تعاملی):

۱. **انتخاب پلتفرم رله**: Vercel یا Netlify
۲. **وارد کردن دامنه**: دامنه‌ای که از قبل رکورد A آن را به سرور تنظیم کرده‌اید
۳. **ایمیل**: برای دریافت گواهی SSL
۴. **تعداد کانفیگ‌ها**: مثلاً ۵ عدد
۵. **حجم هر کانفیگ**: مثلاً ۱۰ گیگابایت (یا ۰ برای نامحدود)
۶. **تنظیمات مسیر و پورت**: می‌توانید پیش‌فرض‌ها را بپذیرید (Enter بزنید)
۷. **توکن پلتفرم**: توکن Vercel یا Netlify خود را وارد کنید
۸. **نصب پنل**: در انتها از شما پرسیده می‌شود که آیا پنل تحت وب نصب شود یا خیر (توصیه: بله)

## 📦 خروجی نهایی

پس از اتمام موفقیت‌آمیز نصب، اطلاعات زیر به شما نمایش داده می‌شود:

- **تمامی لینک‌های کانفیگ** (آمادهٔ کپی در کلاینت)
- **آدرس پنل تحت وب** (مثلاً `https://yourdomain.com:2053`)
- **نام کاربری و رمز عبور پنل**
- **مسیر ذخیرهٔ کانفیگ‌ها** (`/etc/nikvpn/configs.txt`)

نمونه لینک کانفیگ:

```
vless://uuid@relay.vercel.app:443?encryption=none&security=tls&...&path=/api#NikVPN-uuid
```

## 🛠️ مدیریت با ابزار `nikvpn`

پس از نصب، یک ابزار خط‌فرمان با نام `nikvpn` در مسیر `/usr/local/bin/nikvpn` نصب می‌شود. هر زمان که نیاز داشتید می‌توانید از آن استفاده کنید:

```bash
nikvpn configs      # نمایش تمام کانفیگ‌ها
nikvpn panel        # نمایش اطلاعات ورود به پنل
nikvpn status       # بررسی وضعیت سرویس
nikvpn restart      # راه‌اندازی مجدد سرویس
nikvpn add          # راهنمای افزودن کاربر جدید (از طریق پنل)
nikvpn help         # راهنمای کلی
```

## 🖥️ پنل تحت وب (3X-UI)

پنل 3X-UI روی **پورت ۲۰۵۳** با همان گواهی SSL دامنه شما اجرا می‌شود. برای دسترسی به آن:

```
https://yourdomain.com:2053
```

- **نام کاربری**: `admin` (یا مقداری که نصب‌کننده تولید کرده)
- **رمز عبور**: رمز تصادفی نمایش‌داده‌شده در انتهای نصب

### امکانات پنل:

- مشاهده مصرف هر کاربر
- افزودن/حذف کاربران جدید
- تغییر UUID، حجم و تاریخ انقضا
- مشاهده نمودار ترافیک و لاگ‌ها
- تنظیمات پیشرفته Xray

## 📁 فایل‌ها و مسیرهای مهم

| مسیر | توضیح |
|------|-------|
| `/etc/nikvpn/state.env` | فایل ذخیرهٔ وضعیت نصب |
| `/etc/nikvpn/configs.txt` | تمام لینک‌های کانفیگ |
| `/usr/local/bin/nikvpn` | ابزار مدیریت خط‌فرمان |
| `/usr/local/etc/xray/config.json` | تنظیمات Xray |
| `/etc/ssl/xhttp/yourdomain/` | گواهی‌های SSL |
| `/tmp/nikvpn-install.log` | گزارش کامل نصب |

## ⚠️ نکات مهم

- **قبل از نصب**، حتماً رکورد DNS دامنه را به آی‌پی سرور تنظیم کنید و از باز بودن پورت‌های ۸۰ و ۴۴۳ اطمینان حاصل کنید.
- **پورت ۲۰۵۳** برای پنل استفاده می‌شود. اگر فایروال دارید، این پورت را نیز باز کنید.
- **Vercel Deployment Protection** باید **غیرفعال** باشد، در غیر این صورت رله پاسخ ۴۰۱ می‌دهد.
- در صورت استفاده از **Netlify**، مطمئن شوید که سایت به درستی Deploy شده باشد.
- رمز عبور پنل را در جایی امن نگه دارید. در صورت فراموشی، می‌توانید با دستور `nikvpn panel` دوباره آن را ببینید.
